# Music Files - Download Instructions

## Quick Download (5 minutes)

Download these **FREE Public Domain** files and save them here:

### Method 1: Direct Download (Recommended)

Right-click each link → Save As → Save to this folder:

1. [Ambient J Thoughtful](https://archive.org/download/freepd/Page2/Ambient%20J%20Thoughtful.mp3) → Save as `peaceful_morning.mp3`
2. [Chill Deep](https://archive.org/download/freepd/Page2/Chill%20Deep.mp3) → Save as `flowing_water.mp3`
3. [New Age A Weathered](https://archive.org/download/freepd/Page2/New%20Age%20A%20Weathered.mp3) → Save as `mountain_breeze.mp3`
4. [Slow Ticking Clock](https://archive.org/download/freepd/Page2/Slow%20Ticking%20Clock.mp3) → Save as `inner_peace.mp3`

### Method 2: Terminal Download (Mac/Linux)

Run this in your terminal from the project root:

```bash
cd assets/audio/music

curl -L "https://archive.org/download/freepd/Page2/Ambient%20J%20Thoughtful.mp3" -o "peaceful_morning.mp3"
curl -L "https://archive.org/download/freepd/Page2/Chill%20Deep.mp3" -o "flowing_water.mp3"
curl -L "https://archive.org/download/freepd/Page2/New%20Age%20A%20Weathered.mp3" -o "mountain_breeze.mp3"
curl -L "https://archive.org/download/freepd/Page2/Slow%20Ticking%20Clock.mp3" -o "inner_peace.mp3"
```

### Method 3: Pixabay (Alternative)

Visit https://pixabay.com/music/search/meditation/ and download any 4 tracks you prefer.

## After Downloading

1. Verify you have 4 MP3 files in this folder
2. Run `flutter pub get` (if needed)
3. Launch the app
4. Go to Practice tab and test the music player

Done!
